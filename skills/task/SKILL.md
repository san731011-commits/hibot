# Task Skill for OpenClaw
# GitHub 이슈 자동 생성

## Usage

```
/task 제목 | 내용
```

## Examples

```
/task 알림 UI 개선 | 시간 표시 및 모바일 레이아웃 수정
/task API 연동 | 밴드 데이터 fetch 함수 작성
```

## Implementation

This skill creates GitHub issues that can be picked up by Codex on Windows.

Script location: `scripts/create_issue.sh`
